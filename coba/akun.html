Tutorial Penggunaan

1

Membuat laporan harian

Buka menu Input Laporan, isi data cash, saldo, transaksi nasabah, dan mutasi. Simpan laporan untuk menyimpan data harian.

2

Menambahkan transaksi nasabah

Di halaman Input Laporan, tambahkan transaksi nasabah dengan memilih jenis transaksi, nominal, dan admin. Data akan tersimpan otomatis.

3

Mengelola database transaksi

Database menyimpan semua data laporan, pelanggan, jenis transaksi, dan sumber saldo. Gunakan menu Database untuk mengelola data master.

4

Melihat laporan dan analisis

Menu Report menampilkan laporan keuangan lengkap, termasuk cash flow, ringkasan transaksi, dan analisis pendapatan per periode.

5

Melihat rekap bulanan

Rekap Bulanan menyajikan ringkasan keuangan per bulan, termasuk total pemasukan, pengeluaran, saldo, dan grafik tren transaksi.

Tentang Aplikasi

Digital Report dibuat untuk membantu agen transaksi digital, loket pembayaran, dan usaha layanan keuangan harian dalam mencatat aktivitas usaha secara lebih terstruktur.

Aplikasi ini dirancang agar pemilik usaha dapat memantau transaksi, pendapatan, saldo, serta laporan keuangan harian dan bulanan dalam satu sistem yang sederhana dan mudah digunakan.

Tujuan utama aplikasi ini adalah membantu proses pencatatan menjadi lebih rapi, cepat, dan akurat sehingga pemilik usaha dapat memahami kondisi usahanya setiap saat.
